# 🚀 Akash Mishra's AI-First Portfolio

This is a premium, high-performance portfolio website built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**. It is designed to highlight your expertise in Growth Strategy and AI Product Engineering.

## ✨ Features
- **Modern UI**: Dark mode with glassmorphism and subtle gradients.
- **Responsive**: Fully optimized for mobile and desktop.
- **Animated**: Smooth entries and interactive hover states.
- **SEO Optimized**: Pre-configured title tags and meta descriptions.

## 🛠️ Local Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## ☁️ Vercel Deployment (Easy Strategy)

This project is pre-configured for **Vercel**. To deploy:

### Option 1: Vercel CLI (Fastest)
1. Install the Vercel CLI: `npm i -g vercel`
2. Run `vercel` in this directory.
3. Follow the prompts (Select "Yes" to all defaults).

### Option 2: GitHub (Recommended for Long-term)
1. Create a new repository on GitHub.
2. Push this folder to GitHub.
3. Go to [Vercel](https://vercel.com/new) and import the repository.
4. Vercel will automatically detect Next.js and deploy it.

## 📄 Content Management
The content is stored directly in `src/app/page.tsx`. To update your experience or projects, simply edit the arrays at the top of the `Home` component.
